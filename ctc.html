Notes

Why the code is structured this way

`ctc-extraction.js` uploads PDFs to the backend, `functions/ctc-extract.js` extracts embedded text and form fields locally before sending compact document context to AI, and `ctc-ui.js` handles upload state, rendering, validation, and debug output. That keeps the UI stable while we improve extraction providers over time.
